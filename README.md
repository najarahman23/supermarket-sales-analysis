# Supermarket Sales Analysis (SQL + Power BI)

This project analyzes supermarket sales data using **MySQL** for data cleaning & analysis and **Power BI** for interactive dashboard development.  
The dataset was sourced from **Kaggle** and contains 1,000 sales transactions with details on branches, product lines, customer info, payments, and ratings.
